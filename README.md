## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

>### Prerequisites
 - CMD
 - Node.js
>### Setup

- Clone this [repository](git@github.com:taldr27/Stock-Prices.git) to your desired folder.
- Access `cd stock-prices`

>### Install

- Write `npm install`

>### Usage

- Execute `npm start`

>### Run tests

- Execute `npm test`
